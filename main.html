<amp-img> generator online tool developed by Desi Listing. Use this amp-img generator to generate HTML code of valid amp-img tag.

This Tool is useful for blogger user, who have implemented AMP (Accelerated Mobile Pages) framework and find it time consuming to create <amp-img> tag manually.

You might have know that AMP framework doesn't allow use of <img> tag instead you have to change all the <img> tag to <amp-img>.

How to use amp-img generator: Type in URL of your image in ’Image URL Field’. Then type in width and height of your image. Check the box next to your desired layout of your image. And fill Alt, Title and caption field if you want to. After that click get code, and you are done. Copy the image code below ’get code’ button and paste in your webpage.
